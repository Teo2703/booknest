Mini Online Bookstore Static Website Prototype

Open index.html in a browser to view the website.

Pages included:
- index.html
- books.html
- book-detail.html
- register.html
- login.html
- cart.html
- checkout.html
- order-history.html
- group.html
- admin-dashboard.html
- manage-books.html
- manage-orders.html

Notes:
- Static HTML/CSS prototype only.
- No Bootstrap, Tailwind, Foundation, or Bulma is used.
- Final Section B can convert these pages to PHP and connect them to MySQL using XAMPP.
